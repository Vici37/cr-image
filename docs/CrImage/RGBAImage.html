    class CrImage::RGBAImage

    • CrImage::RGBAImage
    • CrImage::Image
    • Reference
    • Object

    Overview

    An image with red, green, blue, and alpha color channels (i.e. a color image). This image type is likely the one read from and written to file (or IO).
